YT-config-generator

Заполните параметры теста, загрузите файл спецификации OpenAPI и скачайте готовый архив настроек для Yandex.Tank.

1. Спецификация
Форматы JSON или YAML.
IP-адрес или домен с портом, куда Yandex.Tank будет отправлять запросы.
Если указан, добавит заголовок Authorization: Bearer {ваш_токен}.
2, Профиль нагрузки
Например: 60s, 5m, 1h. По умолчанию: 60s.
Каким типом измерять поток запросов.
Форма графика изменения нагрузки в ходе теста.
Для 'const'.
Для 'line' и 'step'.
Для 'line' и 'step'.
Только для 'step'.
Используется при типе нагрузки 'Instances'.
3. Авто-остановка теста (Autostop)
Прервать тест раньше времени, если тестируемый сервис начал ломаться.
1. Критерии по HTTP-кодам
Код или семейство кодов (например: 5xx, 404).
Количество или процент ошибок (например: 25, 10%).
Период времени для проверки лимита (например: 10s).
2. Критерии по Сетевым ошибкам
Сетевой код ошибки (например: xx, 110).
Допустимое число сетевых сбоев.
Период времени для проверки лимита (например: 10s).
3. Критерии по времени ответа (Latency)
Остановить тест, если время ответа выше (например: 1500ms, 2s).
Как долго время ответа должно превышать лимит.
