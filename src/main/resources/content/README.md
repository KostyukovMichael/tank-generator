# Настройки нагрузочного тестирования Yandex.Tank

Архив содержит необходимые конфигурационные файлы для проведения тестирования вашего сервиса.

## Быстрый запуск

### Для Linux / macOS:

1. Откройте терминал в папке с распакованными файлами.
2. Выдайте скрипту права на исполнение:
   ```bash
   chmod +x run.sh
   ```
3. Запустите тестирование:
   ```bash
   ./run.sh
   ```

### Для Windows:

1. Убедитесь, что приложение Docker Desktop запущено.
2. Дважды кликните по файлу `run.bat`

---

## Ручной запуск

```bash
docker run \\
  -v $(pwd):/var/loadtest \\
  -it yandex/yandex-tank
```
