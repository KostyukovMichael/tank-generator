# YT-config-generator

Инструмент для быстрого создания конфигурационных файлов **Yandex.Tank** на основе спецификации **OpenAPI (Swagger)**.

---

## Как запустить генератор

Для запуска генератора понадобится установленный Docker (Docker Desktop на Windows).

1. Запустите приложение с помощью скрипта:

```bash
chmod +x scripts/run_with_build.sh
./scripts/run_with_build.sh
```
(Или вручную: `docker build -t tankgen:1.0.0 . && docker run --rm -p 8000:8000 tankgen:1.0.0`)

2. Откройте браузер по адресу: **http://localhost:8000/ui**

3. Заполните форму: прикрепите OpenAPI-файл вашего сервиса, введите желаемые значения параметров теста и нажмите *Сгенерировать*.

4. Распакуйте скачанный ZIP-архив и запустите тест одним из скриптов run.sh/run.bat.

5. Для анализа графиков теста перейдите по адресу **http://localhost:3000**. Появится окно входа в Grafana: введите `login: admin`, `password: admin` (смените его, если информация о графиках конфиденциальна). После чего слева выберите окно `dashboards` и единственный доступный экран - графики получены.
